# Wsparcie

## Jak zgłaszać problemy i uzyskać pomoc

Ten projekt używa GitHub Issues do śledzenia błędów i próśb o funkcje. Proszę przeszukać istniejące
problemy przed zgłoszeniem nowych problemów, aby uniknąć duplikatów. W przypadku nowych problemów zgłoś błąd lub
prośbę o funkcję jako nowy Issue.

Aby uzyskać pomoc i pytania dotyczące korzystania z tego projektu, **OPIEKUN REPOZYTORIUM: Proszę otworzyć problem w celu uzyskania pomocy lub skontaktuj się z właścicielem repozytorium.**
